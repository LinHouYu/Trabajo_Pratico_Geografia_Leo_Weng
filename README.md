## 🇪🇸 Español

**Trabajo Práctico de Geografía: Investigación y Exposición de un Circuito Productivo**

Este proyecto es una presentación web interactiva y altamente visual sobre el circuito productivo del **Té de Roca de Wuyi (China)**. Está desarrollado exclusivamente con HTML, CSS y JavaScript puro, sin depender de librerías externas o software de presentaciones tradicional.

**Características principales:**
* **Efecto 3D Parallax:** Fondo inmersivo de bosque de bambú combinado con un sistema de partículas (Canvas) que simula hojas cayendo dinámicamente.
* **Diseño Glassmorphism:** Interfaz moderna con tarjetas translúcidas de efecto vidrio esmerilado.
* **Interacción Tilt 3D:** Las tarjetas y las fotografías se inclinan suavemente en 3D siguiendo el movimiento del cursor.
* **Traducción en tiempo real:** Botón integrado para cambiar instantáneamente todo el contenido entre Español y Chino.
* **Mapa interactivo integrado:** Integración de Google Maps en vivo para mostrar la localización exacta y los puertos de exportación.
* **Efecto Lightbox:** Todas las imágenes se pueden ampliar fluidamente al hacer clic sobre ellas.

## 🇨🇳 中文

**地理学科实践作业：产业回路的调查与展示**

本项目是一个关于 **中国武夷岩茶** 产业回路的交互式网页演示文稿。项目摒弃了传统的 PPT，完全使用原生 HTML、CSS 和 JavaScript 纯手写开发，带来了极具视觉冲击力的演示体验。

**核心亮点：**
* **3D 视差背景：** 随鼠标移动的竹林背景，搭配原生 Canvas 实时渲染的动态落叶特效，意境十足。
* **玻璃拟态 (Glassmorphism)：** 现代感极强的半透明磨砂玻璃卡片与超大圆角设计。
* **裸眼 3D 悬浮交互：** 所有卡片与素材图会根据鼠标轨迹，通过复杂的数学计算产生 3D 倾斜透视效果。
* **无缝双语切换：** 专为展示设计的左下角按钮，支持一键中西双语实时无缝翻译切换。
* **交互式地图直连：** 网页内嵌 Google Maps 实时卫星地图窗口，精准定位产区与出口港口。
* **灯箱特效 (Lightbox)：** 所有高质量图片素材均支持点击平滑放大查看细节。

## 🇬🇧 English

**Geography Practical Work: Research and Exhibition of a Productive Circuit**

This project is an interactive, highly visual web presentation showcasing the productive circuit of **Wuyi Rock Tea (China)**. It is built entirely from scratch using vanilla HTML, CSS, and JavaScript, offering a modern alternative to traditional slide decks.

**Key Features:**
* **3D Parallax Background:** An immersive bamboo forest background paired with an HTML5 Canvas particle system simulating dynamically falling leaves.
* **Glassmorphism UI:** Modern aesthetic featuring translucent, frosted-glass presentation cards with smooth rounded corners.
* **3D Tilt Interaction:** Presentation cards and images smoothly tilt in 3D space based on the user's mouse movement.
* **Real-time Translation:** Built-in toggle button to switch the entire content seamlessly between Spanish and Chinese.
* **Live Interactive Map:** Embedded Google Maps interface to precisely pinpoint the geographical region and major export ports.
* **Lightbox Effect:** All high-quality images feature a smooth click-to-zoom functionality for detailed viewing.
